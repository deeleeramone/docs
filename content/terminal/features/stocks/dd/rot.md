# Due Diligence

## rot

### Description: 

Rating over time (monthly). [Source: Finnhub]

### Usage: 
```python
usage: rot [-l LIMIT] [--raw]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of last months | 10 | False | None |
| raw | Only output raw data | None | False | None |

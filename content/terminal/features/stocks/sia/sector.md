# Sector Industry Analysis

## sector

### Description: 

See existing sectors, or set sector if arg specified

### Usage: 
```python
usage: sector [-n NAME [NAME ...]]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| name | sector to select | None | False | None |

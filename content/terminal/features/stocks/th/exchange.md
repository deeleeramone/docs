# Trading Hours

## exchange

### Description: 

Select the exchange you want see open hours for

### Usage: 
```python
usage: exchange [-n EXCHANGE]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| exchange | Exchange short name | None | True | None |

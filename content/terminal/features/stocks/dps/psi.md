# Darkpool Shorts

## psi

### Description: 

Shows price vs short interest volume. [Source: Quandl/Stockgrid]

### Usage: 
```python
usage: psi [--nyse]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| b_nyse | Data from NYSE flag. Otherwise comes from NASDAQ. Only works for Quandl. | None | False | None |

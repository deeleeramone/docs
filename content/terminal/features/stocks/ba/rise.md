# Behavioural Analysis

## rise

### Description: 

Print top rising related queries with this stock's query. [Source: Google]

### Usage: 
```python
usage: rise [-l LIMIT]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | limit of top rising related queries to print. | 10 | False | None |

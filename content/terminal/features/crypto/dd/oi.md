# Due Diligence

## oi

### Description: 

Displays open interest by exchange for a certain asset [Source: https://coinglass.github.io/API-Reference/]

### Usage: 
```python
usage: oi [-i {0,1,2,4}]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| interval | Frequency interval. Default: 0 | 0 | False | 0, 1, 2, 4 |

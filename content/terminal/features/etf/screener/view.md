# Screener

## view

### Description: 

View available presets under presets folder.

### Usage: 
```python
usage: view [-p {etf_config.ini}]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| preset | View specific custom preset |  | True | etf_config.ini |

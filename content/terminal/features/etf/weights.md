# ETFs

## weights

### Description: 

Look at ETF sector holdings

### Usage: 
```python
usage: weights [-m MIN] [--raw]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| min | Minimum positive float to display sector | 5 | False | None |
| raw | Only output raw data | None | False | None |

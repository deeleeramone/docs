# Mutual Funds

## sector

### Description: 

Show fund sector weighting.

### Usage: 
```python
usage: sector [-m MIN]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| min | Minimum positive float to display sector | 5 | False | None |

# Futures

## curve

### Description: 

Display futures curve. [Source: YahooFinance]

### Usage: 
```python
usage: curve -t Futures symbol
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| ticker | Future curve to be selected | None | True | None |

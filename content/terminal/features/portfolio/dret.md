# Portfolio

## dret

### Description: 

Daily returns

### Usage: 
```python
usage: dret [-p {3y,5y,10y,all}]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| period | Period to select start end of the year returns | all | False | 3y, 5y, 10y, all |

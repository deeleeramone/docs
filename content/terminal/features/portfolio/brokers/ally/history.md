# Ally

## history

### Description: 

Account transaction history

### Usage: 
```python
usage: history [-l LIMIT]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of recent transactions to show | 15 | False | None |

# paexport



### Usage 
```python
usage: paexport -s START [-e END] [-c CURRENCY]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| start | Start date. | None | False | None |
| end | End date. | 2022-11-18 09:43:58.091097 | True | None |
| currency | Used currency. | USD | True | None |

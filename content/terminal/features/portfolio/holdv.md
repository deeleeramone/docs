# Portfolio

## holdv

### Description: 

Display holdings of assets (absolute value)

### Usage: 
```python
usage: holdv [-u]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| unstack | Sum all assets value over time | None | False | None |

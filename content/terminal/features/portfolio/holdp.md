# Portfolio

## holdp

### Description: 

Display holdings of assets (in percentage)

### Usage: 
```python
usage: holdp [-u]
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| unstack | Sum all assets percentage over time | False | True | None |

# to

Select the "to" currency symbol in a forex pair

### Usage 
```python
usage: to -n TO_SYMBOL
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| to_symbol | To currency | None | False | None |

# Oanda

## from

### Description: 

Select the "from" currency symbol in a forex pair

### Usage: 
```python
usage: from -n FROM_SYMBOL
```

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| from_symbol | From currency | None | True | None |
